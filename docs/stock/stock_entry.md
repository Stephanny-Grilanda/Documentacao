# Entrada no Estoque

Para acessar as telas de controle de estoque siga os passos abaixo:

- Clique na guia **`Estoques`**
- Controle
- Entrada no Estoque

![Aba Controle de Estoque](/img/images/entrada_estoque.png)

Veja abaixo como funciona cada um deles.

:::info

## Pré-requisitos

Para realizar alterações é necessário preencher os requisitos abaixo:

- Ter acesso à página de estoques
- Ter permissão de escrita à página de estoques
  :::

## Entrada no Estoque

A entrada no estoque ocorre automaticamente após o teste de funcionalidade. Para isso, o administrador precisa configurar os destinos no teste e vinculá-los tanto ao estoque quanto aos produtos que serão cadastrados e enviados ao estoque posteriormente.
